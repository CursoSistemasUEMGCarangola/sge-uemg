# ATUALIZAR LESSONS LEARNED

Execute o **Protocolo Learn**:

* Analise o erro ou o requisito obscuro que acabamos de corrigir e a solução aplicada.

* Formate isso como uma nova entrada no `lessons_learned.md` seguindo o template padrão.

* Use a tag adequada (ex: [BUG], [CONFIG]).

* Salve o arquivo.
